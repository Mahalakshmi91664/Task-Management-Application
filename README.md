# 📝 TaskFlow - Task Management System

TaskFlow is a minimalist web application built with **Flask** to help users organize their daily tasks.

## ✨ Features
* **Authentication:** Login and Registration system.
* **Dashboard:** Overview of Total, Pending, and Completed tasks.
* **Management:** Add and track tasks with priority levels.

## 🛠️ Tech Stack
* **Backend:** Python (Flask)
* **Database:** SQLite
* **Frontend:** HTML, CSS, JavaScript
